Welcome to my portfolio. My name is Mercedes Targa Chase-Kelly, I am an aspiring Software Engineer from London.
I am currently undergoing the Full-Stack Codecademy course. The languages I have studied so far are HTML,
CSS and Javascript. I have completed a number of small projects within this course, they can be accessed below.

I studied I.T. in college for three years and my passion for coding was born. My very first lesson was in Procedural
Programming, it consisted of minimal coding in a console using C# but I was enthralled. My passion was further ignited when
I had my first lesson in Event Driven Programming. For my final assignment I created a small program that depicted a fast-food
menu, enabling users to place, edit and submit orders. That was the moment when I knew I wanted to become a Software Engineer.
